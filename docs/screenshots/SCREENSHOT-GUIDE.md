## Screenshots to take (3 required)

### Screenshot 1 — Main chat panel
**File to replace:** `docs/screenshots/screenshot-1-chat.svg`
**File to save as:** `docs/screenshots/screenshot-1-chat.png`

**What to show:**
1. Open Chrome and go to any article page (e.g. a Wikipedia article)
2. Press `Ctrl+E` to open the extension side panel
3. Type a message: *"Summarize this article in 3 bullet points"*
4. Wait for the AI to respond
5. Capture the browser window with the side panel open on the right

**Ideal state:** AI response visible, streaming cursor blinking, side panel and webpage both in frame.

**How:** In ShareX, press `PrintScreen` → select region → select the Chrome window → save as PNG.

**Recommended resolution:** 1280 × 720 minimum.

## GIF to create (1 required — the hero demo)

**File to save as:** `docs/screenshots/demo.gif`
**Replace line in README:** The static banner becomes an animated GIF.

### What to record (30-45 seconds total):

| Seconds | What happens |
|---|---|
| 0–5 | Open a news article page in Chrome |
| 5–8 | Press `Ctrl+E` — side panel slides open |
| 8–18 | Type "Summarize this article" → see AI stream the response |
| 18–28 | Type "Click the read more link" → blue glow activates, phantom cursor moves, click fires |
| 28–38 | New page loads → panel stays open → conversation continues |
| 38–45 | Open settings briefly → show provider list → close |

### How to record with ScreenToGif:

1. Open **ScreenToGif** → click **Recorder**
2. Drag the recording frame to cover your Chrome window (include the side panel)
3. Set **FPS to 15** (File → Options → Recorder → 15 fps — keeps file size small)
4. Click **Record** → do the flow above → click **Stop**
5. In the editor: click **File → Save As → GIF**
6. Under Encoding: set **Quality to 80%**, check **Repeat: Forever**
7. Click Save

**Target file size:** Under 5 MB (GitHub README loads slowly above that).

### Optimise the GIF (if > 5 MB):

    # Install gifsicle (Windows via Chocolatey):
    choco install gifsicle

    # Optimise:
    gifsicle --optimize=3 --colors 128 docs/screenshots/demo.gif -o docs/screenshots/demo.gif
